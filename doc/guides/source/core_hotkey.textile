h2. Aloha HotKeys

WARNING: The HotKey feature is still being developed. Not all shortcuts are available at the moment.


|_. Document action |_. Shortcut |
| Select all | Ctrl+A |
| Copy | Ctrl+C |
| Print | Ctrl+P |
| Save | Ctrl+S |
| Paste | Ctrl+V |
| Cut | Ctrl+X |
| Undo | Ctrl+Z |
| Redo | Ctrl+Y |
| Show your browser's right-click options | Shift+right-click |
| Increase paragraph indentation | Tab |
| Decrease paragraph indentation | Shift+Tab |
| Insert footnote | Ctrl+Alt+F |
| Insert comment | Ctrl+Alt+M |


|_. Text formatting |_. Shortcut |
| Bold | Ctrl+B |
| Italicize | Ctrl+I |
| Underline | Ctrl+U |
| Superscript | Ctrl+. |
| Subscript | Ctrl+, |
| Copy formatting | Ctrl+Alt+C |
| Paste formatting | Ctrl+Alt+V |
| Clear formatting | Ctrl+\ |
