fillTuples ( maps in module aloha )

Description

Fills the given map with entries from the given tuples.

Parameters

  • map – {Object} –

    The given map will have one entry added for each item in the given array.

  • tuples – {Array[Array[String, *]]} –

    An array of [key, value] tuples. Javascript maps can only contain string keys, so the keys must be strings or or they will be cast to string.

Returns

{Object} –

The given map.
