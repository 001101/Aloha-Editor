  8 ['aloha', 'aloha/jquery', 'aloha/plugin', 'aloha/floatingmenu', 'i18n!characterpicker/nls/i18n', 'i18n!aloha/nls/i18n', 'css!characterpicker/css/characterpicker.css'],
  9 function(Aloha, jQuery, Plugin, FloatingMenu, i18n, i18nCore) {

 13 		GENTICS = window.GENTICS;

 15 	function CharacterOverlay(characters, onSelectCallback) {
 16 		var self = this;
 17 		self.$node = jQuery('<table class="aloha-character-picker-overlay" role="dialog"><tbody></tbody></table>');
 18 		// don't let the mousedown bubble up. otherwise there won't be an activeEditable 
 19 		self.$node.mousedown(function(e) {
 20 			return false;
 21 		});
 22 		self.$tbody = self.$node.find('tbody');
 23 		self.$node.appendTo(jQuery('body'));
 24 		self._createCharacterButtons(self.$node, characters, onSelectCallback);
 25 		self._initHideOnDocumentClick();
 26 		self._initHideOnEsc();
 27 		self._initCursorFocus(onSelectCallback);
 28 		self._initEvents();
 29 	}
 30 	CharacterOverlay.prototype = {
 31 		show: function( insertButton ) {
 32 			var self = this;
 33 			// position the overlay relative to the insert-button
 34 			self.$node.css(jQuery(insertButton).offset());
 35 			self.$node.show();
 36 			// focus the first character
 37 			self.$node.find('.focused').removeClass('focused');
 38 			jQuery(self.$node.find('td')[0]).addClass('focused');
 39 		},
 40 		_initHideOnDocumentClick: function() {
 41 			var self = this;
 42 			// if the user clicks somewhere outside of the layer, the layer should be closed
 43 			// stop bubbling the click on the create-dialog up to the body event
 44 			self.$node.click(function(e) {
 45 				e.stopPropagation();
 46 			});
 47 			// hide the layer if user clicks anywhere in the body
 48 			jQuery('body').click(function(e) {
 49 				var overlayVisibleAndNotTarget
 50 					=  (self.$node.css('display') === 'table')
 51 					&& (e.target != self.$node[0])
 52 				    // don't consider clicks to the 'show' button.
 53 					&& !jQuery(e.target).is('button.aloha-button-characterpicker');
 54 				if(overlayVisibleAndNotTarget) {
 55 					self.$node.hide();
 56 				}
 57 			});
 58 		},
 59 		_initHideOnEsc: function() {
 60 			var self = this;
 61 			// escape closes the overlay
 62 			jQuery(document).keyup(function(e) {
 63 				var overlayVisibleAndEscapeKeyPressed = (self.$node.css('display') === 'table') && (e.keyCode === 27);
 64 				if(overlayVisibleAndEscapeKeyPressed) {
 65 					self.$node.hide();
 66 				}
 67 			});
 68 		},
 69 		_initCursorFocus: function( onSelectCallback ) {
 70 			var self = this;
 71 			// you can navigate through the character table with the arrow keys
 72 			// and select one with the enter key
 73 			var $current, $next, $prev, $nextRow, $prevRow;
 74 			var movements = {
 75 				13: function select() {
 76 					$current = self.$node.find('.focused');
 77 					self.$node.hide();
 78 					onSelectCallback($current.text());
 79 				},
 80 				37: function left() {
 81 					$current = self.$node.find('.focused');
 82 					$prev = $current.prev().addClass('focused');
 83 					if($prev.length > 0) {
 84 						$current.removeClass('focused');
 85 					}
 86 				},
 87 				38: function up() {
 88 					$current = self.$node.find('.focused');
 89 					$prevRow = $current.parent().prev();
 90 					if($prevRow.length > 0) {
 91 						$prev = jQuery($prevRow.children()[$current.index()]).addClass('focused');
 92 						if($prev.length > 0) {
 93 							$current.removeClass('focused');
 94 						}
 95 					}
 96 				},
 97 				39: function right() {
 98 					$current = self.$node.find('.focused');
 99 					$next = $current.next().addClass('focused');
100 					if($next.length > 0) {
101 						$current.removeClass('focused');
102 					}
103 				},
104 				40: function down() {
105 					$current = self.$node.find('.focused');
106 					$nextRow = $current.parent().next();
107 					if($nextRow.length > 0) {
108 						$next = jQuery($nextRow.children()[$current.index()]).addClass('focused');
109 						if($next.length > 0) {
110 							$current.removeClass('focused');
111 						}
112 					}
113 				}
114 			};
115 			jQuery(document).keydown(function(e) {
116 				e.stopPropagation( );
117 				var isOverlayVisible = self.$node.css('display') === 'table';
118 				if(isOverlayVisible) {
119 					// check if there is a move-command for the pressed key
120 					var moveCommand = movements[e.keyCode];
121 					if(moveCommand) {
122 						moveCommand();
123 						return false;
124 					}
125 				}
126 			});
127 		},
128 		_initEvents: function() {
129 			var self = this;
130 			// when the editable is deactivated, hide the layer
131 			Aloha.bind('aloha-editable-deactivated', function(event, rangeObject) {
132 				self.$node.hide();
133 			});
134 		},
135 		_createCharacterButtons: function($node, characters, onSelectCallback) {
136 			var self = this;
137 			function htmlEntityToSingleCharacter(character) {
138 				// isn't there any better way?
139 				var textarea = document.createElement('textarea');
140 				textarea.innerHTML = character;
141 				return textarea.value;
142 			}
143 			function mkButton(c) {
144 				var character = htmlEntityToSingleCharacter(c)
145 				return jQuery('<td>' + character + '</td>')
146 					.mouseover(function() {
147 						jQuery(this).addClass('mouseover');
148 					})
149 					.mouseout(function() {
150 						jQuery(this).removeClass('mouseover');
151 					})
152 					.click(function(e) {
153 						self.$node.hide();
154 						onSelectCallback(character);
155 						return false;
156 					});
157 			}
158 			function addRow() {
159 				return jQuery('<tr></tr>').appendTo(self.$tbody);
160 			}
161 			var characterList = jQuery.grep(
162 				characters.split(' '),
163 				function filterOutEmptyOnces(e) {
164 					return e != '';
165 				}
166 			);
167 			var i=0, char;
168 			var $row;
169 			while(char = characterList[i]) {
170 				// make a new row every 15 characters
171 				if(((i%15)===0)) {
172 					$row = addRow();
173 				}
174 				mkButton(char).appendTo($row);
175 				i++;
176 			}
177 		}
178 	};
179

193 			}
194 			var insertButton = new Aloha.ui.Button({
195 				'name': 'characterpicker',
196 				'iconClass': 'aloha-button-characterpicker',
197 				'size': 'small',
198 				'onclick': function(element, event) { self.characterOverlay.show(element.btnEl.dom); },
199 				'tooltip': i18n.t('button.addcharacter.tooltip'),
200 				'toggle': false
201 			});
202 			FloatingMenu.addButton(
203 				'Aloha.continuoustext',
204 				insertButton,
205 				i18nCore.t('floatingmenu.tab.insert'),
206 				1
207 			);

210 		onCharacterSelect: function(character) {
211 			var self = this;
212 			var range = Aloha.Selection.getRangeObject();
213 			if(Aloha.activeEditable) {
214 				var charNode = jQuery(document.createTextNode(character));
215 				GENTICS.Utils.Dom.insertIntoDOM(
216 					charNode,
217 					range,
218 					jQuery(Aloha.activeEditable.obj),
219 					true
220 				);
221 				range.select();
222 			}
223 		}
224 	});
