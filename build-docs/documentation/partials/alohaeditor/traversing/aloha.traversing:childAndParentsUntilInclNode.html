childAndParentsUntilInclNode ( traversing in module aloha )

Description

Collects the given node, and all its ancestors until untilInclNode is reached.

Parameters

  • node – {DOMObject} –

    given node

  • untilInclNode – {DOMObject} –

    Terminal ancestor. Will be included in results.

Returns

{Array[DOMObject]} –

A set of parent element of the given node.
