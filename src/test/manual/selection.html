Aloha Editor developers testbox

Add the markup you want to test and hit Strg+ENTER. Selection markers are not allowed as text. If you want to test a certain selection use following syntax to mark selection begin and end.
[{ data-start only one occurrence allowed. Defines begin of the selection.
]} data-end only one occurrence allowed. Defines end of the selection.
