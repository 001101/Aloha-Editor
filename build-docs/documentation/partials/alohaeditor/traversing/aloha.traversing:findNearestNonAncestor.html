findNearestNonAncestor ( traversing in module aloha )

Description

Returns the nearest (in the document order) to the given node that is not an ancestor of that node.

Parameters

  • node – {DOMObject} –

    given node

  • previous – {Boolean} –

    If true, will look for the nearest preceeding node, otherwise the nearest subsequent node.

  • limit – {Function(DOMObject):Boolean} –

    (Optional) Predicate, which will be applied to each node in the traversal step. If this function returns true, traversal will terminal and findNearestNonAncestor will return null.

Returns

{DOMObject} –

nearest non ancestor node
