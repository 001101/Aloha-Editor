skipUnrenderedToEndOfLine ( html in module aloha )

Description

Tries to move the given point to the start of the line, stopping to the right of a br or block node, ignoring any unrendered nodes. Returns true if the point was successfully moved to the start of the line, false if some rendered content was encountered on the way. point will not be mutated unless true is returned.

Parameters

  • point – {Cursor} –

    given cursor point

Returns

{Boolean} –

True if the cursor is moved.
