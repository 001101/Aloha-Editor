equal ( arrays in module aloha )

Description

Does a shallow compare of two arrays.

Parameters

  • a – {Array} –

    An array to compare.

  • b – {Array} –

    A second array to compare with a .

  • equalFn – {Function(*, *):Number} –

    A custom comparison function that accepts two values a and b from the given arrays and returns true or false for equal and not equal respectively.

    If no equalFn is provided, the algorithm will use the strict equals operator.

Returns

{Boolean} –

True if all items in a and b are equal, false if not.
