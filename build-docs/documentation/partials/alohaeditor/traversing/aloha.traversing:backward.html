backward ( traversing in module aloha )

Description

Given a node, will return node that preceeds it in the document order.

For example, if this function is called recursively, starting from the text node "five" in the below DOM tree:

"one" "two" "three" "four" "five"

backward() will return nodes in the following order:

"four", "three", ..., ..., "two", ..., "one"

Parameters

  • node – {DOMObject} –

    given node

Returns

{DOMObject} –

The preceeding node or null if the given node has no previous siblings and no parent.
