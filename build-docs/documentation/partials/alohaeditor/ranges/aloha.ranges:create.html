create ( ranges in module aloha )

Description

Creates a range object with boundaries defined by containers and offsets in those containers.

Parameters

  • startContainer – {DOMElement} –

    a startContainer

  • startOffset – {Number} –

    a startOffset

  • endContainer – {DOMElement} –

    an endContainer

  • endOffset – {Number} –

    an endOffset

Returns

{Range} –

range object with boundaries
