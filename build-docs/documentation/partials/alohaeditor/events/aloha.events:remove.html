remove ( events in module aloha )

Description

Detaches the specified event callback from the given event.

Parameters

  • obj – {DOMObject|Document|Window} –

    Object which supports events. This includes DOM elements, the Document itself, and the Window object for example.

  • event – {String} –

    Name of the event to detach.

  • callback – {Function} –

    Function to be de-registered.
