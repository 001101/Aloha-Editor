climbUntil ( traversing in module aloha )

Description

Climbs up the given node's ancestor until the predicate until() return true. Starting with the given node, applies func() to each node in the climb.

Parameters

  • node – {DOMObject} –

    given node

  • func – {Function(DOMObject)} –

    function applied for each node

  • until – {Function(DOMObject):Boolean} –

    until that function returns true
