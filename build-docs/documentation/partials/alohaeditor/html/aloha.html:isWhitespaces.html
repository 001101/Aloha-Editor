isWhitespaces ( html in module aloha )

Description

Checks whether or not a given text node consists of only sequence of white space characters as defined by W3 specification:

http://www.w3.org/TR/html401/struct/text.html#h-9.1

Parameters

  • textnode – {DOMElement} –

    a given text node

Returns

{boolean} –

True is node is a textnode of white characters.
