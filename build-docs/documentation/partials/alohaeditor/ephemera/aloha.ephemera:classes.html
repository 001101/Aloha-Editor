classes ( ephemera in module aloha )

Description

Registers ephemeral classes.

An ephemeral class is a non-content class that will be pruned from the from the result of editable.getContents().

The given classes should contain the string 'aloha-' to get the benefit of a performance optimization.

Returns a map that contains all classes that were ever registered with this function.

Multiple classes may be specified. If none are specified, just returns the current ephemeral classes map without modifying it.

Also see ephemera().
