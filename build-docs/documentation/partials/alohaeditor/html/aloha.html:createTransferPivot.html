createTransferPivot ( html in module aloha )

Description

Returns an object containing the properties start and move .

start a node that is visually (ignoring any unrendered nodes inbetween) to the right node .

move is function that will correctly move nodes from right to left (right of node ) starting from start , all the way to the visual end of the line.

Parameters

  • node – {DOMObject} –

    The node that is on the left side of the join.

Returns

{Object} –

contains the properties start and move
