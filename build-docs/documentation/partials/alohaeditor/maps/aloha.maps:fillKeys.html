fillKeys ( maps in module aloha )

Description

Fills the given map with the given keys mapped to the given value.

Parameters

  • map – {Object} –

    The given map will have one entry added for each given key.

  • keys – {Array[String]} –

    An array of string keys. JavaScript maps can only contain string keys, so these must be strings or they will be cast to string.

  • value – {String} –

    A single value that each given key will map to.

Returns

{Object} –

The given map.
