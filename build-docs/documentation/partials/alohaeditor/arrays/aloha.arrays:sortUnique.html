sortUnique ( arrays in module aloha )

Description

Implements unique() using native sort().

Parameters

  • a – {Array} –

    The array to sort and strip of duplicate values. Warning: this array will be modified in-place.

  • compFn – {Function(*, *):Number} –

    A custom comparison function that accepts two values a and b from the given array and returns -1, 0, 1 depending on whether a < b, a == b, a > b respectively.

    If no compFn is provided, the algorithm will use the browsers default sort behaviour and loose comparison to detect duplicates.

Returns

{Array} –

The given array, sorted and with duplicates removed.
