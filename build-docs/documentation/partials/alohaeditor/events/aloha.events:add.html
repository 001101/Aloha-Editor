add ( events in module aloha )

Description

Registers an event listener to fire the given callback when a specified event is triggered on the given object.

Parameters

  • obj – {DOMObject|Document|Window} –

    Object which supports events. This includes DOM elements, the Document itself, and the Window object for example.

  • event – {String} –

    Name of the event for which to register the given callback

  • callback – {Function} –

    Function to be invoked when event is triggered on the given object.
