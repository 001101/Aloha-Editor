hasInlineStyle ( html in module aloha )

Description

Checks whether the given node is rendered with inline style.

An inline node is a node that is not a block node.

Note that this function depends on style inheritance which only works if the given node is attached to the document.

Parameters

  • node – {DOMObject} –

    given node

Returns

{Boolean} –

True if the given node is rendered with inline style.
