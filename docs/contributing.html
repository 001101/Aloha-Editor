Contributing

to Aloha Editor

Thank you for taking an interest in contributing to the Aloha Editor project.

In order to develop Aloha Editor as open source project in a proper legal way, all contributors need to sign a contribution agreement similar to the one from the Apache Software Foundation, if their contributions should be included into the core code base.

Only pull requests of contributors who have signed the CLA may be merged.

If you want to contribute please read the Aloha Editor contribution agreement and send a proper filled and signed copy to
