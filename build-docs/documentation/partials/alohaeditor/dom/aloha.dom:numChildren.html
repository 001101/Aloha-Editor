numChildren ( dom in module aloha )

Description

Calculates the number of child nodes contained in the given DOM element.

elem.childNodes.length is unreliable because "IE up to 8 does not count empty text nodes." (http://www.quirksmode.org/dom/w3c_core.html)

Parameters

  • elem – {DOMObject} –

    given element

Returns

{Number} –

Number of children contained in the given node.
