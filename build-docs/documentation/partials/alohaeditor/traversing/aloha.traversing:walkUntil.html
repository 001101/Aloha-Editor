walkUntil ( traversing in module aloha )

Description

Applies the given function func() , to the the given node node and it's next siblings, until the given until() function retuns true or all next siblings have been walked.

Parameters

  • node – {DOMObject} –

    given node

  • func – {Function(DOMObject, *)} –

    Callback function to apply to the traversed nodes. Will receive the each node as the first argument, and the value of arg as the second argument.

  • until – {Function(DOMObject, *):Boolean} –

    Predicate function to test each traversed nodes. Walking will be terminated when this function returns true . Will receive the each node as the first argument, and the value of arg as the second argument.

  • arg – {*} –

    A value that will be passed to func() as the second argument.
