allowsNesting ( content in module aloha )

Description

Checks whether the node name outer is allowed to contain in a node with the node name inner as a direct descendant based on the HTML5 specification.

Reference: http://www.w3.org/html/wg/drafts/html/master/index.html#elements-1 http://www.whatwg.org/specs/web-apps/current-work/#elements-1

Parameters

  • outer – {String} –

    The node which would contain the other.

  • inner – {String} –

    The node to be nested a child of outer .

Returns

{Boolean} –

True if inner is allowed a direct child of outer .
