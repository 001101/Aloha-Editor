forward ( traversing in module aloha )

Description

Given a node, will return node that succeeds it in the document order.

For example, if this function is called recursively, starting from the text node "one" in the below DOM tree:

"one" "two" "three" "four" "five"

forward() will return nodes in the following order:

..., "two", ..., ..., "three", "four", "five"

Parameters

  • node – {DOMObject} –

    given node

Returns

{DOMObject} –

The succeeding node or null if the given node has no previous siblings and no parent.
