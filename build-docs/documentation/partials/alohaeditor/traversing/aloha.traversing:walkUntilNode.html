walkUntilNode ( traversing in module aloha )

Description

Applies the given function func() , to the the given node node and it's next siblings, until untilNode is encountered or the last sibling is reached.

Parameters

  • node – {DOMObject} –

    given node

  • fn – {Function(DOMObject, *)} –

    Callback function to apply to the traversed nodes. Will receive the each node as the first argument, and the value of arg as the second argument.

  • untilNode – {DOMObject} –

    Terminal node.

  • arg – {*} –

    A value that will be passed to func() as the second argument.
