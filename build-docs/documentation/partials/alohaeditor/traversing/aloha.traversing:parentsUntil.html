parentsUntil ( traversing in module aloha )

Description

Traverses up the given node's ancestors, collecting all parent nodes, until the given predicate returns true.

Parameters

  • node – {DOMObject} –

    given node

  • pred – {Function(DOMObject):Boolean} –

    Predicate function which will receive nodes as they are traversed. This function returns true , it will terminate the traversal.

Returns

{Array[DOMObject]} –

A set of parent elements of the given node.
