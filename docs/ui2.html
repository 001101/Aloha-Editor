Leap 1: Include and invoke Aloha

To replace ContentEditable with Aloha Editor is as simple as 1 line of JavaScript: 

aloha(document.querySelector('#editable'))

You can initialize multiple editables at once with this simply trick: 

aloha.dom.query('.editable', document).forEach(aloha)

aloha() transforms the given element into an editing canvas; you can start typing and using keyboard commands to format content bold, italic, etc.

To disable editing use aloha.mahalo()

Leap 2: Add a button

Aloha Editor uses a blinking DIV element as the caret. It is the minimal visual user interface (which you can style with CSS btw!). But you'll want more than just a caret. So this is how you can a bold to format content button with Aloha Editor.

Add this markup into your document. 

<button class="my-big-bold-button">B</button>
NB: We're using jQuery API as a shorthand because it's terse and everyone's familliar with it; you can pick your own poison. 
$('.my-big-bold-button').on('click', aloha.ui.command(aloha.ui.commands.bold))

Leap 3: Add many buttons

There a number of formatting commands that come with Aloha Editor out of the box. You saw the aloha.ui.commands.bold command, but there are a few more.

Let's use a for-loop to bind 4 commands to buttons. 

<button class="action-bold">B</button>
<button class="action-italic">I</button>
<button class="action-underline">U</button>
<button class="action-unformat">âœ˜</button>

for (var command in aloha.ui.commands) {
  $('.action-' + command).on('click', aloha.ui.command(aloha.ui.commands[command]));
}

Leap 4: Per-editable binding

But what if we want to the underline button to be able to effect one editable and not the other?

To do this we need to pass 2 arguments into aloha.ui.command instead of 1. You'll see why in a minute. First let add some suffixes to our button classnames. We add 1 to demark buttons that should effect editable 1 and 2 to demark buttons that should effect editable 2. 

<button class="action-bold-1">B</button>
<button class="action-underline-2">U</button>

var editables = aloha.dom.query('.editables', document).map(aloha);
for (var editable in editables) {
  for (var command in aloha.ui.commands) {
    $('.action-' + command + '-' + editable.id).on(
      'click',
      aloha.ui.command(editable, aloha.ui.commands[command])
    );
  }
}

Bare in mind that aloha.ui.command can also take an array of editables as its first argument to bind a single button to more than just one editables.
