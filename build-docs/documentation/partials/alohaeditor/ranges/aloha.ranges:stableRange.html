stableRange ( ranges in module aloha )

Description

Creates a "stable" copy of the given range.

A native range is live, which means that modifying the DOM may mutate the range. Also, using setStart/setEnd may not set the properties correctly (the browser may perform its own normalization of boundary points). The behaviour of a native range is very erratic and should be converted to a stable range as the first thing in any algorithm.

Parameters

  • range – {Range} –

    given range

Returns

{StableRange} –

a stable range
