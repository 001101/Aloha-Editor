nextWhile ( traversing in module aloha )

Description

Starting from the given node and moving forward, traverses the set of node 's sibiling nodes until either the predicate cond returns false or we reach the last sibling of node 's parent element.

Parameters

  • node – {DOMObject} –

    given node

  • cond – {Function(DOMObject, *?):Boolean} –

    given condition

  • arg – {*} –

    Optional arbitrary value that will be passed to the cond() predicate.

Returns

{DOMObject} – node , or one if it's next siblings.
