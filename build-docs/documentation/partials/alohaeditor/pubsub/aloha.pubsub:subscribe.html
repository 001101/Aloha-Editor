subscribe ( pubsub in module aloha )

Description

Subscribes a callback function to a channel. Whenever this channel publishes, this function will be invoked. The return value is an id which identifies this subscription (a channel, and callback tuple). This id can be used to unsubscribe this subscription from the given channel.

Parameters

  • channel – {String} –

    Name of channel to listen on.

  • callback – {Function(Object)} –

    Function to be invoked when messages are published on the given channel.

Returns

{Number} –

Positive integer representing the sid of this subscription, that can be used with unsubscribe() if subscription succeeds. Otherwise the return value is -1;
