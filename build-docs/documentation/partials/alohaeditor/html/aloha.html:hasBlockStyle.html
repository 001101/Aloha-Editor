hasBlockStyle ( html in module aloha )

Description

Checks whether the given node is rendered with block style.

A block node is either an Element whose "display" property does not have resolved value "inline" or "inline-block" or "inline-table" or "none", or a Document, or a DocumentFragment.

Note that this function depends on style inheritance which only works if the given node is attached to the document.

Parameters

  • node – {DOMObject} –

    given node

Returns

{Boolean} –

True if the given node is rendered with block style.
