outparameter ( functions in module aloha )

Description

Creates a bound variable and returns the closure which can be used to get and set the value of it as a free variable.

This construct can be used to as a convenient way to simulate generic out parameters in JavaScript.

Calling the closure with an argument changes the value of the enclosed variable. Calling the closure without any arguments will return the value of the enclosed variable.

Parameters

  • value – {*} –

    The initial value that the enclosed variable should hold.

Returns

{Function(*):*} –

closure of variable
