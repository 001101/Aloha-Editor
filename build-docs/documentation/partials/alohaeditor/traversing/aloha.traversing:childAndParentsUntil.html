childAndParentsUntil ( traversing in module aloha )

Description

Collects all ancestors of the given node until the first ancestor that causes the given predicate function to return true.

Parameters

  • node – {DOMObject} –

    given node

  • pred – {Function(DOMObject):Boolean} –

    Predicate function which will receive nodes as they are traversed. This function returns true , it will terminate the traversal.

Returns

{Array[DOMObject]} –

A set of parent element of the given node.
